Module 02: The Tactical Suite

1. Speed 1: "Tab" (The Ghost)

Copilot ++. Real-time code prediction.
Unlike traditional autocomplete, it predicts your cursor movement and edits.

Advanced Tab Techniques:
  • Use Ctrl + ‚Üí (Right Arrow) to accept suggestions word-by-word. Great when the AI is 90% right but messes up the last variable.
  • Cpp (Cursor Prediction Mode): If enabled, it attempts to predict the next change you make after you move your cursor, not just new code.

2. Speed 2: "Cmd+K" (The Scalpel)

The Inline Edit prompt. Shortcut: Ctrl+K (Win) / Cmd+K (Mac).

Process for .NET Developers:

  1. Highlight a C# Method.
  2. Press Ctrl+K.
  3. Prompt: "Convert this foreach loop to a LINQ expression."
  4. Review the Diff:
    • Red Background: Lines being removed.
    • Green Background: Lines being added.

3. Speed 3: "Chat" (The Brain)

The Sidebar Interface (Ctrl+L). Use this for planning or debugging.

Common Mistake: Do NOT use Chat for small edits. Copy-pasting code from Chat to your file breaks your "Flow State". Use Cmd+K for edits. Use Chat for Questions.

4. Advanced Technique: The "Fix & Refactor" Loop

Scenario: Legacy C# Code Cleanup

Step 1 (Chat):
Ask: "How can I modernize this C# 7.0 code to use C# 12 features like primary constructors?"

Step 2 (Cmd+K):
Highlight the class. Prompt: "Apply Primary Constructors and use file-scoped namespace as discussed."

Step 3 (Tab):
Go to the properties. Start typing validatiors. Tab will auto-complete standard Guard clauses.
