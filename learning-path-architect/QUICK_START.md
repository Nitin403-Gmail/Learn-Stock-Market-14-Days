## 5-Minute Setup

### Step 1: Access Copilot Studio
1. Go to [copilotstudio.microsoft.com](https://copilotstudio.microsoft.com)
2. Sign in with your Microsoft 365 account

### Step 2: Create Agent
1. Click **"Create"** or **"New agent"**
2. In **"Describe your agent to create it"** field, paste:
   ```
   Learning Path Architect creates personalized learning journeys for career development. 
   It assesses current skills, identifies growth opportunities, creates customized learning 
   paths aligned with career goals, curates high-quality resources from diverse sources, 
   tracks progress, and adapts learning plans dynamically.
   ```

### Step 3: Add Description
- In description field: `Learning Path Architect creates personalized learning journeys for career development`

### Step 4: Create & Test
1. Click **"Create"** (wait 2-5 minutes)
2. Click **"Test your agent"**
3. Try: `"Hi, I want to start my learning journey"`

### Step 5: Enable Work Context (Optional)
1. Go to **Settings** ‚Üí **Generative AI** ‚Üí **Work Context**
2. Enable **"Use work context"** toggle
3. Configure data sources (Emails, Documents, Meetings)

## Full Instructions Setup

For the complete agent with all features, after initial creation:

1. Go to **Settings** ‚Üí **Generative AI** ‚Üí **Instructions**
2. Copy the full instruction block from `readme.md` (lines 10-519)
3. Paste into the instructions field
4. Save and test

## Troubleshooting

| Issue | Quick Fix |
|-------|-----------|
| Agent not created | Check Copilot license, try shorter description first |
| No M365 data | Enable "Work Context" in settings |
| Generic responses | Grant permissions, check work context enabled |
| Resources not found | Add knowledge sources in settings |
